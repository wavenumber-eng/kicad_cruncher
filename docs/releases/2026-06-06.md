# KiCad Cruncher 2026-06-06

Release version: `2026.6.6`

This release publishes the first public KiCad IPC plugin and daemon framework
for `kicad-cruncher`. It keeps the existing manufacturing and SVG/STEP command
surface while adding the installable plugin path needed for appz to delegate
KiCad plugin setup to the public package.

## Highlights

- Adds `kicad-cruncher plugin list-targets`, `plugin install`, `plugin status`,
  and `plugin uninstall` for bundled KiCad IPC plugin packages.
- Adds the `kicad-cruncher-tools` KiCad IPC action package.
- Adds a local daemon with `/health`, `/version`, command inventory, daemon
  state discovery, and PCB clean planning routes.
- Routes PCB clean through CLI, daemon, and plugin adapters over shared cleanup
  logic.
- Adds a KiCad IPC apply adapter for cleanup mutation requests under KiCad
  commit/undo behavior.
- Preserves appz setup as a thin workspace wrapper while removing appz-owned
  KiCad plugin implementation, installer, assets, and plans.

## Plugin Install Notes

The public plugin command installs into KiCad user plugin folders and can report
or update KiCad IPC API preferences. Use `--enable-api` while KiCad is closed
when setup should enable KiCad's IPC server preference. Restart KiCad after
installation so plugin discovery and toolbar construction run from the new
settings.

The plugin action currently opens the local KiCad Cruncher daemon tool center
and routes PCB clean requests through the daemon. It does not auto-start the
daemon yet.
