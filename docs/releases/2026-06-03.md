# KiCad Cruncher 2026-06-03

Release version: `2026.6.3`

This release publishes `kicad-cruncher` against `kicad-monkey==2026.6.3`.

## Highlights

- Adds the `pcb-svg` preview command for A0 PCB layer outputs and configured
  design views.
- Adds pin-1 marker views, assembly HLR/bounds overlays, assembly designator
  overlays, muted assembly copper colors, and smoother derived board-outline
  arc sampling.
- Keeps `wn-geometer==2026.5.25` as the controlled assembly HLR dependency.

## Preview Notice

`pcb-svg` is a preview feature in this release. The SVG structure, virtual-layer
metadata, default view set, and `pcb.svg.config` controls are expected to evolve
after more real-world board testing. Treat generated SVG/config details as
subject to change in future releases.

## Signoff

- Uses the date-versioned package contract for `2026.6.3`.
- Pins `kicad-monkey==2026.6.3` and `wn-geometer==2026.5.25`.
