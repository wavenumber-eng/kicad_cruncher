# KiCad Cruncher 2026-06-04

Release version: `2026.6.4`

This release publishes `kicad-cruncher` with initial KiCad manufacturing
output support while continuing to use `kicad-monkey==2026.6.3`.

## Highlights

- Adds initial `bom`, `pnp`, and `jlc` commands for KiCad projects and boards.
- Adds a shared documented `bom.config` JSONC contract for BOM/PnP/JLC outputs,
  including field alias coalescing, variant selection, DNP handling, grouped BOM
  columns, PnP columns, and output path templates.
- Adds JLCPCB upload outputs: grouped BOM XLSX and CPL XLSX using KiCad
  aux-axis/drill-place-origin-relative component-center coordinates.
- Adds `pcb-layer-step` fixture-alignment STEP output for KiCad PCB layers with
  configurable copper, pads, vias, drills, board outlines, and fused copper
  review bodies.
- Keeps `pcb-svg` as a preview feature while expanding real-board coverage for
  configured views, assembly overlays, virtual layers, and designator rendering.

## Manufacturing Support

The BOM/PnP/JLC commands are initial public support. The config and output
shape are now documented and covered by release tests, but real-world board
feedback may still drive compatible additions in future releases.

PnP and JLC CPL coordinates currently use the KiCad footprint placement point as
the documented `component-center` mode. The implementation intentionally does
not expose alternate centroid or center-of-mass modes yet.

## Preview Notice

`pcb-svg` remains a preview feature in this release. The SVG structure,
virtual-layer metadata, default view set, and `pcb.svg.config` controls may
evolve after more real-world board testing.

## Signoff

- Uses the date-versioned package contract for `2026.6.4`.
- Pins `kicad-monkey==2026.6.3` and `wn-geometer==2026.6.4`.
- CI and release workflows run Rack, build, Twine check, and installed-console
  smoke tests before publishing.
