# KiCad Cruncher 2026-05-31

Release version: `2026.5.31`

This is the initial public repository setup for `kicad-cruncher`.

## Highlights

- Adds the package scaffold for a `uv tool install kicad-cruncher` CLI.
- Adds the first public command, `design`, which emits KiCad-native design JSON
  through the public `kicad-monkey==2026.5.31` API.
- Adds `pcb-svg` for A0 PCB SVG layer outputs and configured design views,
  including `wn-geometer==2026.5.25` assembly HLR overlays.
- Adds GitHub CI, PyPI trusted publishing workflow, Rack strata, design-doc
  signoff, py_signoff, ruff, pyright, build, twine, and installed-console
  checks.
