CLI command design

design

Generates machine-consumable KiCad-native design JSON from KiCad project or schematic inputs.

Usage

kicad-cruncher design project.kicad_pro -o output/design
kicad-cruncher design schematic.kicad_sch
kicad-cruncher design --no-indexes

Arguments

file accepts .kicad_pro or .kicad_sch. When omitted, the command attempts project auto-detection when exactly one .kicad_pro exists in the current directory. -o selects the output directory, and --no-indexes omits lookup indexes.

Output

The command writes KiCadDesign.to_json() output to <input-stem>_design.json. The payload is owned by kicad-monkey and includes project metadata, schematic hierarchy, sheets, components, nets, variants, optional PnP data, and optional lookup indexes.

Tests

L3 verifies command execution against a redistributable synthetic KiCad project and asserts the design JSON schema, project text variables, component list, net list, and index suppression behavior.
